This is the NED mobile client

NED mobile client utilizes the following external project components. We have specified the location, the name (e.g. .jar name), given a short description and the software license: 
1. java/lib/
       kxml2-2.3.0.jar - kXML is a lean Common XML API with namespace and WAP support - The content of the license is compliant with MIT license.
       detailed license: licenses/kxml2_license.txt
2. java/src/org/ned/client/utils
       BasicAuth.java - basic authentication source code - BSD license.
       detailed license: licenses/basicauth_license.txt

For more information about NED, please see http://projects.forum.nokia.com/NED
